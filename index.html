    How to get equipment Type B?

    When killing any monster, starting from the "Sette Desert" location, there is a chance to get a box with weapon or armor, which drop a weapon Type B or armor Type B for your own race only. Boxes stack in inventory to save space.
    Equip level close to minimum monster level =>[Open spoiler with examples] Examples of matching monster level and equipment level:
    Example 1: Caliana Warrior (Eter) is most relevant for gaining experience for a character of level 37-40, so when she is killed, there is a chance to get a level 40 weapon, as well as a 37-39 armor or shield.
    Example 2: Giant Baba (Cauldron) is most relevant for gaining experience for a level 46-49 character, so when he is killed, there is a chance to get a level 45-50 weapon, as well as a 47-50 armor or shield.
    Example 3: Traitor (Sette Desert) is most relevant for gaining experience for a level 33-36 character, so when he is killed, there is a chance to get a level 35 weapon, as well as a 33-35 armor or shield.

    How to create weapons Type C and Type D grade?

    To improve the grade of equipment, you need to combine at NPC Hero "Main item" and "Material".

    The main item for creating a Type C weapon is a Type B weapon. The material is a Type A weapon, not modified with talics. Type A weapons from level 35 to level 70 are sold by NPCs at the HQ.

    The main item for crafting a Type D weapon is a Type C weapon. The material is Leon Tokens. Leon tokens can be obtained for completing any quests and dungeons from level 50 to 70. WARNING: Type D weapons and Leon's Tokens are non-tradable items.

    Combination Features:
    The "Main Item" cannot be destroyed if the combo fails, only the "Materials".
    "Main item" saves the number of talics in the new item after a successful combo.
    The weapon property Type C depends on the property in the weapon Type A. The property is saved when creating Type D.
